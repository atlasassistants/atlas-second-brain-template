# Sarah Chen's Kill List — Banned Patterns

Patterns banned in Sarah's voice. Drafting skills load this and reject any output containing these patterns.

## Universal banned (ship-with-template)

These banned patterns apply to almost any voice, not just Sarah's. They're the AI-formula tells that read as machine-written.

### Tone moves

- "In today's fast-paced world..." / "In the modern era..." / any throat-clearing essay opener
- "It's important to note that..." / "It's worth mentioning..." / hedging meta-commentary
- "As we navigate..." / "As we look ahead..." / future-gazing without specifics
- "Game-changer" / "leverage" / "synergy" / "robust" / "seamless" / generic business-prose vocabulary

### Punctuation patterns

- Em-dash everywhere (—) as a rhythm crutch in every paragraph
- "Ah, the X..." / "Ah, but..." / playful-AI openings
- Three-period ellipsis as suspense filler (...) when not actually trailing off

### Structural moves

- Numbered lists where the points repeat the same idea three times in different words
- Bold-everything-emphatic where every other sentence has bold
- Parallel-structure-fetish: "X is not Y. X is Z. X is W." beat repeated 4+ times in one piece

### Self-intro warm-up

- "Hi, I'm X..." / "Let me introduce myself..." / "Before we dive in, a quick word about..." — never warm up; open with the hook.

## Sarah-specific kill items

### Tone moves
- "Founder mindset" / "founder energy" — over-used in startup discourse
- "Move fast and break things" — rejected as a cliché
- "First principles thinking" — used too often without depth
- Any framing that romanticizes burnout
- Apologizing in print: "Sorry for the long post" / "TL;DR if you're busy"

### Topics
- Hot-take pieces about competitors (Sarah believes these don't compound)
- Personal-life details unless directly relevant to the build
- Crypto/web3 takes outside of infrastructure context

### Structural
- Longer than 1200 words for newsletter pieces (Sarah keeps it tight)
- More than 3 bullet points in a single section
- Subhead-as-clickbait without paying off
